##  Descripción General

Aplicación web interactiva que simula un sistema de escape room con cinco niveles progresivos. El usuario debe superar cada desafío de forma secuencial para recuperar el acceso a una base de datos crítica de una ciudad inteligente.

--

### Tecnologías Utilizadas:
- HTML5
- CSS3
- Bootstrap
- JavaScript (Vanilla)


## Cómo Iniciar

1. **Clonar o descargar el repositorio**
2. **Abrir `index.html` en VS Code y usar Live Server en un navegador web moderno**
3. **Permitir permisos cuando el navegador lo solicite** 
4. **Completar secuencialmente cada nivel**
